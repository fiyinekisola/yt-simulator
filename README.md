# 📺 YT Simulator

A browser-based YouTube creator simulation game where you upload videos, grow your channel, go viral, get monetised, and try to become the biggest YouTuber on the planet — without getting banned.

Built with React + Vite.

---

## 🎮 How to Play

1. **Upload videos** by choosing a content category (Gaming, Music, Vlogs, etc.)
2. **Grow your subscribers** through views, viral moments, and smart upgrades
3. **Apply for monetisation** once you hit 1,000 subscribers
4. **Spend your earnings** on upgrades to grow faster
5. **Apply for verification** at 100,000 subscribers
6. **Reach 100,000,000 subscribers** to become a LEGEND

---

## ⚡ Features

- 🎬 **10 content categories** — each with different view and subscriber multipliers
- 🔥 **Viral system** — rare chance to blow up, scales as your channel grows
- 💀 **Flop system** — videos can bomb, especially early on
- 💵 **Monetisation gate** — apply to the YouTube Partner Programme at 1K subs
- ✅ **Verification gate** — apply for the checkmark at 100K subs
- 🎵 **Copyright claims** — labels can claim your video revenue as you grow
- 💣 **Buy subscribers** — risky black market growth with suspension/ban consequences
- ⚠️ **Suspension system** — get caught buying subs and face appeals, lost verification, and a 10% re-verify rate
- 🔴 **Permanent ban** — push your luck too far and it's game over
- 🛒 **10 upgrades** — better camera, SEO mastery, brand agency and more
- 📊 **Stats & milestones** — track every number on your road to LEGEND

## 🚀 Getting Started

### Prerequisites
- Node.js (LTS) — [nodejs.org](https://nodejs.org)
- Git — [git-scm.com](https://git-scm.com)

### Installation

```bash
git clone https://github.com/fiyinekisola/yt-simulator.git
cd yt-simulator
npm install
npm run dev
```

Then open [http://localhost:5173](http://localhost:5173) in your browser.

### Build for production

```bash
npm run build
```
